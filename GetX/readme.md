> <a href="https://pub.dev/packages?q&sort=like">pub.dev에서 가장 많은 좋아요를 받은 패키지</a> (2024년에도 건재하다)

소규모 개발에서 간단한 사용법으로 정말 유용하게 사용했던 프레임워크,  
`상태 관리, 라우팅, 의존성 관리` 의 3가지 주요 기능 뿐만 아니라,  
`테마, 반응형 크기` 등도 GetX를 사용하여 쉽게 적용할 수 있다.

## 목차 (내맘대로)

- 주요 기능 3가지
   1. 라우팅
   2. 상태 관리
   3. 의존성 관리
- 부가 기능
  - 위젯
  - 테마
  - 미디어쿼리
  - 다국어
  - (그외 기능들은 공식문서 참고)
