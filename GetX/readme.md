소규모 개발에서 간단한 사용법으로 정말 유용하게 사용했던 프레임워크,  
`상태 관리, 라우팅, 의존성 관리` 의 3가지 주요 기능 뿐만 아니라,  
`테마, 반응형 크기` 등도 GetX를 사용하여 쉽게 적용할 수 있다.

## 목차 (내맘대로)

1. 주요 기능
   1. 상태 관리
   2. 라우팅
   3. 의존성 관리
2. 잡기술
   1. 테마
   2. 반응형 크기
