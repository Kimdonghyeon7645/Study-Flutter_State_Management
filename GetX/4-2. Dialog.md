## Get.defaultDialog

- 간단하게 다이얼로그를 생성 가능

```dart
OutlinedButton(
onPressed: () => Get.defaultDialog(
    title: "경고",
    middleText: "경고는 영어로 Warning",
    textConfirm: "확인",
    onConfirm: () => print("ok"),
    // textCancel: "취소",  // 취소버튼도 추가 가능
  ),
  child: Text("버튼"),
),
```

## Get.dialog

- 직접 위젯으로 만들어서 생성 가능

```dart
Get.dialog(
  Dialog(
    surfaceTintColor: Colors.white,
    child: Container(
      // ... (다이얼로그 위젯 정의)
    ),
  ),
)
```
